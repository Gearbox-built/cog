# Cog

Script to automate lots of bash stuff

`Usage: cog <command>`

## Prerequisites

### 1. Install/Update requirements

- rvm - `rvm get head`

### 2. Setup bash alias

Run the following to setup your `cog` alias:

```sh
echo "alias cog='bash ~/scripts/gearbox-cog/cog.sh'" >> ~/.bash_profile && source ~/.bash_profile
```
